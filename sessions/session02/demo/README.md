- **Shortcuts:**
  - Duplicate line: `Shift` + `Alt` + `↓`
  - Select current line: `Ctrl` + `L`
  - Wrap selection with tag: `Alt` + `Shift` + `W`
  - Select next match: `Ctrl` + `D`
